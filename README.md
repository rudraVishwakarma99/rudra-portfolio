# Rudra Kumar Vishwakarma | Portfolio

A modern, responsive portfolio website built with React, TypeScript, Vite, and Sass. This project showcases frontend engineering experience, technical skills, and professional highlights in a clean, recruiter-friendly layout.

## Features
- Responsive portfolio layout
- Theme toggle support
- Filterable experience timeline by skill
- Built with React + TypeScript
- Deployed on GitHub Pages

## Tech Stack
- React
- TypeScript
- Vite
- Redux Toolkit
- Sass

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
```bash
npm run deploy
```
